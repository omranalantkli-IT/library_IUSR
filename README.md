\# Library Lite (HTML + CSS)



\## فكرة المشروع

واجهة نظام مكتبة خفيف لعرض الكتب والأعضاء والإعارات، بهدف التدريب على:

\- Git / GitHub (Branches, PRs, Commits)

\- Jira (To Do / In Progress / Done)

\- Documentation داخل الـ Repository



> المشروع Front-End فقط (HTML + CSS). لا يوجد حفظ بيانات أو منطق إعارة بدون JavaScript/Backend.



\## أعضاء الفريق والأدوار

\- Member 1: (UI Layout / Navbar / Pages)

\- Member 2: (Books Page + Table)

\- Member 3: (Members + Loans Pages)

\- Member 4: (Documentation + Screenshots + Git Workflow)



\## التقنيات المستخدمة

\- HTML5

\- CSS3 (Responsive + UI Components)



\## تشغيل المشروع

1\. افتح الملف `index.html` مباشرة بالمتصفح.

2\. أو استخدم Live Server في VS Code لعرض الصفحات.



\## صفحات النظام

\- `index.html` لوحة تحكم

\- `books.html` قائمة الكتب

\- `members.html` قائمة الأعضاء

\- `loans.html` الإعارات

\- `add-book.html` إضافة كتاب (واجهة)
